Pipeline de Logística (ETL com Python)

O que o código faz?
O pipeline segue um fluxo simples e eficiente:

Lê os dados: Pega as informações brutas das viagens que estão em um arquivo CSV.

Analisa: Aplica uma regra de negócio que compara o peso da carga com a capacidade máxima do veículo.

Informa: Gera um relatório limpo no terminal, mostrando quem está liberado ("OK") e quem excedeu o limite ("BARRADO").

Por que fiz esse projeto?
Estou estudando Engenharia de Dados e esse foi um exercício prático para entender como funciona um fluxo de ETL (Extração, Transformação e Carga) na vida real. Foi um desafio muito legal aprender a tratar esses dados e transformar uma planilha bruta em informação útil!

Ferramentas que usei:
Python para a lógica do pipeline.

Git e GitHub para controlar as versões e guardar o projeto.

VS Code para escrever todo o código.
